Motivation and context

Well, want to get some advice on studying material around engineering subjects?

You could of course look into a million forums and social media group, but I think, if you really want to get the information from real geeks, just type Github awesome in the browser! There are a ton of lists of awesome material regarding lots of subjects, and this is my personal version of such a list, dive in!

The list itself

  • Industrial products

    • Mcmastercarr the go-to place to find a specific industrial product by image search, ISO specification and so on
    • Fixitech mechanical fastening systems
    • Elesa components for the mechanical industry
    • Kipp components for the mechanical industry
    • Southco industrial latches and Hardware solutions
    • Fastloc industrial latches and Hardware solutions
    • Sugatsune Architectural & Industrial Component Hardware

  • Sheet metal bending

    • Rolleri Sheet metal bending tools

  • Information about metallic industrial designations

    • Steel number

  • CAD repositories

    • GrabCAD
    • PartCommunity

  • Theory reference for engineers

    • Continuum mechanics recap on continuum mechanics
    • Everyspec Repository of standards, this in particular is an Handbook for manual calculation from Nasa

  • News for engineers

    • IMechanica

  • Command line tools

    • ss64 the go to place to find documentation on command line utilities
    • Git most needed version control system

  • Free Open Source Software

    • Arduino Telemetry
    • MixTex Compiler for Latex, the only document writing tool you will use once you know it
    • Gimp Image manipulation program
    • Inkscape Vector image manipulation program
    • QGIS digital topography and much more
    • Anaconda Scientific distribution for python
    • GitJournal This is really a smartphone app but I insist on keeping it here cause it’s such a basic and important tool for keeping notes and thought organized
    • Notepad++ Such a simple and fundamental tool
    • SumatraPDF A basic, distraction free, pdf and ebook reader

  • Open source hardware projects

    • Hackaday
    • Hackster

  • Robotics

    • ROS Robot Operating System
    • Gazebo Robot simulation software
    • Pybullet Real time physics simulation
    • AI for Manufacturing Robotics

  • PhD

    • Euraxess Career center for reseachers in Europe
    • The PhD Lab

  • Patents

    • Espacenet
    • Google patents

  • Interesting Companies

    • Vesuvius Leader in molten metal flow engineering
    • Arrival EV manufacturers, inventors of the Microfactories solution
    • Small Robot Companies Manufacturers of robot solution for agriculture
    • Markforged Industrial additive manufacturing platform
    • Formalabs High Resolution SLA and SLS 3D Printers
    • Roboze Professional and industrial 3D printers and manufacturing as a service
    • Simscale Simulation software reinvented for the web
